# El Gambito de UTEC
Ajedrez en C++ desde la terminal - CS1112 Programación II 2026-I

## Cómo ejecutar en CLion

1. Abrir CLion y seleccionar **Open**, elegir la carpeta del proyecto.
2. CLion detectará el `CMakeLists.txt` automáticamente — confirmar **Trust Project**.
3. Esperar a que CLion indexe el proyecto.
4. Presionar el botón **Run** (triángulo verde) o `Shift + F10`.

## Controles
- Ingresar casillas en formato `ColFila` — ejemplo: `E2`, `G1`.
- Columnas: A–H | Filas: 1–8.
- Las piezas blancas mueven primero.
